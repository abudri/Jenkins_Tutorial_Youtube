## Jenkins on Your Mac

Installed Jenkins using Homebrew here: 

https://www.jenkins.io/download/lts/macos/


### **Start Jenkins Locally**

`$ brew services start jenkins-lts`

### **Visit the UI Locally**
http://localhost:8080/

### Helpful Local Files

